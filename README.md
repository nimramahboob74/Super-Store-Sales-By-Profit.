# Super Store Sales – Profit Analysis

## 📌 About the Project
This project analyzes retail sales data from a superstore to identify 
profit trends across different product categories, regions, and customer 
segments. The goal is to uncover which areas of the business are most 
and least profitable to support better business decisions.

## 🛠️ Tools Used
- Microsoft Excel / Power BI
- Data Analysis & Visualization

## 📊 Dataset Features
| Column | Description |
|---|---|
| Order ID | Unique order identifier |
| Category | Product category (Furniture, Technology, Office Supplies) |
| Sub-Category | Product sub-category |
| Region | Sales region (East, West, Central, South) |
| Sales | Total sales amount |
| Profit | Profit earned per order |
| Discount | Discount applied |
| Quantity | Number of items ordered |

## 🔍 What I Did
- Cleaned and organized raw superstore sales data
- Analyzed profit by product category and sub-category
- Identified top-performing and loss-making regions
- Examined the impact of discounts on overall profitability
- Visualized sales vs profit trends across different segments

## 💡 Key Insights
- Technology category generates the highest profit margin
- Heavy discounts on Furniture lead to significant losses
- The West region consistently outperforms other regions in profit
- Office Supplies have high order volume but lower profit per order
